Asus ProArt PA328Q

The Asus ProArt PA328Q is a 32" monitor boasting Ultra HD resolution (3840 x 2160 pixels). It also features 10-bit color and 100% sRGB all due to its factory pre-calibration for unmatched color accuracy. It also has an ultra-slim profile with ergonomic tilt providing easy adjustments. Run everything in true 4k resolution.

The Asus PA328Q is currently not available but it is expected to arrive some time near the end of this year.
